Australian Climate Data Visualization

Average Maximum Temperature Trend by City

This interactive line chart displays the Average Annual Maximum Temperature trends for selected major Australian cities/stations. Use the dropdown menu on the chart to select and compare trends from different locations.

Map of All Weather Stations by Year

This map shows the location of every weather station in the dataset. The color of each point represents the station's maximum temperature per years, illustrating the climate variation across the country.

Map of All Weather Stations by Average Temperature

This map shows the location of every weather station in the dataset. The color of each point represents the station's overall average maximum temperature across all recorded years, illustrating the climate variation across the country.
